1) 🧠 What is React?
React = UI library (not full framework)
Purpose: build reusable, interactive UI
Component-based
Data-driven → state + props
Virtual DOM → fast updates

Key Idea:

React uses JS + HTML in JSX + state + props → efficient rendering

2) ⚡ JSX Explained

JSX = JavaScript XML → HTML-like syntax inside JS

const element = <h1>Hello, Nirjhor!</h1>;

Rules:

Must return single parent element
Use {} for JS expressions
className instead of class
Inline styles → object {}

Example:

const style = { color: "blue", fontSize: "20px" };
const element = <h1 style={style}>Hello, JS + React!</h1>;
3) ⚡ Functional Components
Function returns JSX
Simple, reusable
Recommended over class components (modern React)
function Welcome(props) {
  return <h2>Hello, {props.name}!</h2>;
}

ReactDOM.render(<Welcome name="Nirjhor" />, document.getElementById("root"));
props = input to component
ReactDOM.render → renders in browser
4) 🔄 Components Composition

Components can contain other components:

function App() {
  return (
    <div>
      <Welcome name="Nirjhor" />
      <Welcome name="Akash" />
    </div>
  );
}
App = parent component
Welcome = child component
Props pass data
