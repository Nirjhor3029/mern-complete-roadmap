## Today's Learning

### Core Concepts

1. **What is React?**
   - UI library (not full framework)
   - Component-based architecture
   - Virtual DOM for efficient updates

2. **JSX**
   - JavaScript XML - HTML-like syntax in JS
   - Must return single parent element
   - Use `{}` for JS expressions
   - `className` instead of `class`

3. **Functional Components**
   - Simple JavaScript functions returning JSX
   - Modern React standard
   - Recommended over class components

4. **Props**
   - Data passed to components
   - Read-only
   - One-way data flow

5. **Component Composition**
   - Components containing other components
   - Parent-child relationship

## Key Takeaways

- React = efficient UI rendering via Virtual DOM
- JSX = declarative syntax combining JS + HTML
- Functional components = modern, recommended approach
- Props = read-only data from parent to child
- Composition = building complex UI from simple components
