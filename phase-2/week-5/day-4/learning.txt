1) 🧠 Why normal variable fails
function Counter() {
  let count = 0;

  function increment() {
    count++;
    console.log(count);
  }

  return <button onClick={increment}>Count: {count}</button>;
}
❌ Problem

button click করলে console এ count বাড়বে,
কিন্তু UI update হবে না.

কারণ:

React শুধু state change detect করলে re-render করে

normal variable track করে না.

2) ⚛️ useState() Basic
import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  function increment() {
    setCount(count + 1);
  }

  return <button onClick={increment}>Count: {count}</button>;
}
✅ Breakdown
const [count, setCount] = useState(0);
count → current state value
setCount → updater function
0 → initial value
3) 🔁 Re-render Flow

When:

setCount(count + 1);

React does:

update state
schedule re-render
call component function again
update UI

এই জন্য function component বারবার re-run হয়.

এখানে JS execution context knowledge directly লাগে.

4) ⚠️ Async / Stale Update Bug

এইটা senior-level important.

function Counter() {
  const [count, setCount] = useState(0);

  function incrementTwice() {
    setCount(count + 1);
    setCount(count + 1);
  }

  return <button onClick={incrementTwice}>{count}</button>;
}
❓ Output?

only +1 😮

কারণ দুইবারই same old count capture করেছে.

এটা হচ্ছে:

stale closure / stale state

JS closure Day-3 এর real React use 🔥

5) ✅ Functional Update Fix
setCount(prev => prev + 1);
setCount(prev => prev + 1);

Full:

function incrementTwice() {
  setCount(prev => prev + 1);
  setCount(prev => prev + 1);
}

এখন output হবে +2 ✅

কারণ React latest state pass করে.

6) 🛒 Real Life Example
Quantity Button
function Product() {
  const [qty, setQty] = useState(1);

  return (
    <div>
      <button onClick={() => setQty(qty - 1)}>-</button>
      <span>{qty}</span>
      <button onClick={() => setQty(qty + 1)}>+</button>
    </div>
  );
}
7) 🎯 State with Objects
const [user, setUser] = useState({
  name: "Nirjhor",
  age: 22
});

Update:

setUser({
  ...user,
  age: user.age + 1
});

এখানে JS spread operator direct লাগছে.
