## Today's Learning

### Core Concepts

1. **Advanced JSX**
   - JavaScript expressions in `{}`
   - Functions, math, ternary inside JSX
   - What doesn't work: statements (if, for, switch)

2. **Conditional Rendering**
   - Ternary Operator: `condition ? A : B`
   - Logical AND: `condition && Component`
   - Function with if/else for complex logic

3. **Rendering Lists**
   - `array.map()` for rendering multiple elements
   - Each list item needs unique `key`

4. **Keys**
   - Unique identifiers for React's reconciliation
   - Use database IDs, not random values
   - Index as last resort only

## Key Takeaways

- JSX allows expressions, not statements
- Ternary for A/B choice, && for show/hide
- Keys are critical for performance
- Filter before map for filtered lists
- Always handle empty states
