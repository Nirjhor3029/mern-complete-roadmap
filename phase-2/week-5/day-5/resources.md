## Quick Reference

| Concept | Key Code |
|---------|----------|
| Controlled input | `<input value={value} onChange={e => setValue(e.target.value)} />` |
| Form submit | `e.preventDefault()` in onSubmit handler |
| Multiple inputs | `...form, [e.target.name]: e.target.value` |
