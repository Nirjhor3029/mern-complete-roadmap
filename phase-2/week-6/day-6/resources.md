## Quick Reference

| Hook Name | Purpose |
|-----------|---------|
| useCounter | Increment/decrement/reset state |
| useInput | Form input handling |
| useFetch | API data fetching |
| useLocalStorage | Persist data to localStorage |
| useToggle | Boolean toggle |
| useDebounce | Debounce value |
