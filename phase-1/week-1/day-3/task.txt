## Task 1: Counter with Reset
```javascript
function createCounter() {
  // Create counter with:
  // - increment(): increases by 1
  // - decrement(): decreases by 1
  // - reset(): sets count back to 0
  // Return an object with these methods
}

const counter = createCounter();
console.log(counter.increment()); // 1
console.log(counter.increment()); // 2
console.log(counter.reset());
console.log(counter.increment()); // 1
```
**Answer:** Use closure to maintain count variable

## Task 3: Predict Output
```javascript
function outer() {
  let x = 10;
  return function() {
    console.log(x);
  };
}
const fn = outer();
x = 20;
fn();
```
**Answer:** 20 (reads current value of x when called)

## Task 4: Loop Closure Fix
```javascript
// Fix this bug using both methods:
// 1. Using let
// 2. Using IIFE

// Method 1 (let):
for (let i = 0; i < 5; i++) {
  setTimeout(() => console.log(i), 100);
}

// Method 2 (IIFE):
for (var j = 0; j < 5; j++) {
  // Your IIFE here
}
```
**Answer:** 
- let: Output 0,1,2,3,4
- IIFE: Wrap in `(function(index){...})(j)`

## Task 5: Currying
```javascript
function multiply(a) {
  // Return a function that takes b and returns a * b
}

const double = multiply(2);
const triple = multiply(3);

console.log(double(5)); // 10
console.log(triple(5)); // 15
```
**Answer:** Return `function(b) { return a * b; }`
